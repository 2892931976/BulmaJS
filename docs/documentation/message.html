Dismissable messages

While messages can be awesome for informing your users of something. There are some cases where you would like to provide the user the ability to close (dismiss) the message. The message plugin adds the ability to dismiss a message.

When creating the message via Javascript you simply add the isDismissable: true property inside the options object. This will automatically add the close button and listen for it's click handler.

Alternativly, if you're using the Data API you can instead add the delete button to the markup, and the plugin will work everything out for you.

Automatically dismiss

There may be a need to automatically dismiss a message. You can easily accomplish this by either specifying a dismissInterval property inside the Javascript options object or by using the data-dismiss-interval when using the Data API. Both of these methods will accept a integer representing the number of milliseconds to wait before hiding the message.

Destroy the element when dismissed

If you want to remove the element from the page when it has been dismissed. You can add the destroyOnDismiss: true property to the Javascript object. Alternativly, you can add the data-destroy attribute to the element via the Data API.

Changing the colour

If you're using Javascript to create your message component, you can pass an additional colour property to the options object. This property will accept a string representing the colour/state you would like the message to take. Just to note, this property is taken as is with the is- prefix added. This means you can provide you're own custom colours to this if you wish.

Changing the size

Similar to how you would go about changing the colour of your message if using the Javascript API, you can also adjust the size. Simply pass a size option to the options object, specifying the size you would like to use.

Methods

Method name Parameters Description
show() None This will show the message (remove the is-hidden class.
hide() None This will hide the message (add the is-hidden class.

Options

Option name Required Default Type Description
body Yes '' String This is the body of the message.
parent No document.body HTMLElement This is the element you would like the message to be appended to.
color No '' String This is the color modifer to add to the message. For example for is-info you would set color to info.
size No '' String This is the size modifer to add to the message. For example for is-large would make the message the larger variant.
dismissInterval No null Integer This is the time in milliseconds the message is visible.
isDismissable No false Boolean Would you like the message to be dismissable, we will also add the delete button to the rendered HTML.
destroyOnDismiss No true Boolean Would you like the message to be destroyed when dismissed? This will remove the HTML and also unregister events.
element No true HTMLElement This is a reference to a .message element. Generally you do not need to include this value, it is primarily used for the automatic DOM parsing.
closeButton No <button class="delete"></button> HTMLElement This is the element you would like to be used when dismissing the message.

Data Attributes

Option name Required Default Type Description
data-bulma Yes Nothing String This is the component type, to use the message component this must be message.
data-dismiss-interval No null Integer The amount of time in milliseconds until the message is automatically dismissed.
