By default the Bulma dropdown component can only be shown by adding a is-active class to the dropdown container. This requires a little bit of Javascript to detect a click and track the current state of the dropdown.

The below example shows an example dropdown component. Along with the example markup. Note that the only thing you need to do is add the data-bulma="dropdown" attribute to the dropdown container and the data-trigger attribute to the button or other element you would like to trigger the dropdown to show or hide. The library will handle the rest.

If you instead want the dropdown to open/close based on the hover state, you can instead add `is-hover` to the dropdown. This removes the need for this plugin.

DOM options

Option name Required Default Type Description
data-bulma Yes Nothing String This is the component type, to enable the dropdown functionality this should be dropdown.
data-trigger Yes Nothing String The element you would like to be used as the trigger. This will assign the click event listener.
