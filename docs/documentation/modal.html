Modals are a super useful component to have at our finger tips. They can be used for a wide range of things, from alert messages to displaying a 'quick view' of content. After you've create the HTML for your modal, you can then jump into the Javascript side of things and bring it to life.

Start by creating the BulmaJS modal instance. Use the Bulma.create method to create a new modal, passing your root modal element as an option.

{% highlight javascript %} {%- include code/modal/create-snippet.js -%} {% endhighlight %}

And that's it, you now have a modal instance. To open the modal, simply call the open method on the modal instance. You can then close the modal by calling the close method.
