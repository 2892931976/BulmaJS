Dismissable notifications

While notifications can be great for notifying your users of something. There are some cases where you would like to provide the user the ability to close (dismiss) the notification. The notification plugin adds the ability to dismiss a notification.

When creating the notification via Javascript you simply add the isDismissable: true property inside the options object. This will automatically add the close button and listen for it's click handler.

Alternativly, if you're using the Data API you can instead add the delete button to the markup, and the plugin will work everything out for you.

{% highlight javascript %} {%- include code/notification/dismissable.js -%} {% endhighlight %}

Automatically dismiss

There may be a need to automatically dismiss a notification. You can easily accomplish this by either specifying a dismissInterval property inside the Javascript options object or by using the data-dismiss-interval when using the Data API. Both of these methods will accept a integer representing the number of milliseconds to wait before hiding the notification.

{% highlight javascript %} {%- include code/notification/auto-dismissing.js -%} {% endhighlight %}

Destroy the element when dismissed

If you want to remove the element from the page when it has been dismissed. You can add the destroyOnDismiss: true property to the Javascript object. Alternativly, you can add the data-destroy attribute to the element via the Data API.

Changing the colour

If you're using Javascript to create your notification component, you can pass an additional colour property to the options object. This property will accept a string representing the colour/state you would like the notification to take. Just to note, this property is taken as is with the is- prefix added. This means you can provide you're own custom colours to this if you wish.

Changing the size

Similar to how you would go about changing the colour of your notification if using the Javascript API, you can also adjust the size. Simply pass a size option to the options object, specifying the size you would like to use.

Methods

Method name Parameters Description
show() None This will show the notification (remove the is-hidden class.
hide() None This will hide the notification (add the is-hidden class.

Javascript options

Option name Required Default Type Description
body Yes '' String This is the body of the notification.
parent No document.body HTMLElement This is the element you would like the notification to be appended to.
color No '' String This is the color modifer to add to the notification. For example for is-info you would set color to info.
dismissInterval No null Integer This is the time in milliseconds the notification is visible.
isDismissable No false Boolean Would you like the notification to be dismissable, we will also add the delete button to the rendered HTML.
destroyOnDismiss No true Boolean Would you like the notification to be destroyed when dismissed? This will remove the HTML and also unregister events.
element No true HTMLElement This is a reference to a .notification element. Generally you do not need to include this value, it is primarily used for the automatic DOM parsing.
closeButton No <button class="delete"></button> HTMLElement This is the element you would like to be used when dismissing the notification.

DOM Options

Option name Required Default Type Description
data-bulma Yes Nothing String This is the component type, to use the notification component this must be notification.
data-dismiss-interval No null Integer The amount of time in milliseconds until the notification is automatically dismissed.
